# Enhanced Dashboard Skill

**Version:** 1.0.0
**Phase:** 9 - Silver Tier
**Status:** Complete

---

## Purpose

Generate real-time dashboard with multi-platform summary, pending approvals, active plans, workflow status, financial tracking, recent activity, and system status.

---

## When to Use

- **Event-driven updates:** When new items arrive from any source
- **Scheduled updates:** Every 60 seconds for real-time monitoring
- **On-demand updates:** When user requests dashboard refresh
- **After workflow completion:** To reflect latest status
- **After financial transactions:** To update budget/expense displays

---

## Features

### 1. Multi-Platform Summary
- Email, WhatsApp, LinkedIn, Files counts
- Table format with item counts per platform
- Real-time aggregation from all sources

### 2. Pending Approvals
- Top 5 items awaiting approval
- Shows: Item ID, Type, Amount, Priority, Deadline
- Sorted by priority and deadline

### 3. Active Plans
- Plans currently in progress
- Progress percentage (completed/total steps)
- Status indicator (active/paused/blocked)

### 4. Workflow Status
- Running and paused workflows
- Current step indicator
- Workflow type and ID

### 5. Financial Tracking
- Pending and paid invoices count
- Total expenses with budget comparison
- Budget percentage and remaining amount
- Alert indicators at 80% threshold

### 6. Recent Activity
- Last 10 actions across all systems
- Timestamped activity log
- Action descriptions

### 7. System Status
- Watcher status (email, whatsapp, linkedin, files)
- Database connection status
- Last backup timestamp

## Usage

```python
from src.skills.enhanced_dashboard import EnhancedDashboard

# Initialize
dashboard = EnhancedDashboard(vault_path)

# Generate dashboard
result = dashboard.generate_dashboard()

# Update on event
result = dashboard.update_on_event('new_item', {'source': 'email'})

# Scheduled update
result = dashboard.scheduled_update()
```
